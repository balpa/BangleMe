# BangleMe

iOS AR app — the user points their phone's camera at their arm, picks a gold bracelet from the bottom carousel, and sees it on their wrist in hyper-realistic detail. A social-media-filter-style entertainment app.

## Status

Foundation in place. Plan 1 (camera + wrist tracking) is implemented and committed; on-device verification pending an iOS platform install. Plans 2–6 still to come.

## Tech Stack

- Swift 5.9 + SwiftUI
- AVFoundation (camera)
- Vision framework (hand / wrist detection)
- RealityKit + Reality Composer Pro (3D render, PBR materials, IBL)
- ReplayKit (video capture)
- Target: iOS 16+, iPhone 12 and up

## Getting Started

1. Install Xcode 15+ (this repo was generated against Xcode 26 / iOS 16+).
2. Install [xcodegen](https://github.com/yonaskolb/XcodeGen) if you need to regenerate the project: `brew install xcodegen`.
3. Make sure the iOS platform component is installed — `xcodebuild -downloadPlatform iOS` or Xcode → Settings → Platforms.
4. Open `BangleMe.xcodeproj`, select your device, ⌘R.
5. Show your wrist to the back camera; a yellow circle should track it.
6. To continue development, open `docs/superpowers/plans/` and follow the next plan task by task.

## Tests

Pure-logic units (pose math, One Euro filter, fade-in/out state) are covered by XCTest and run in the iOS Simulator:

```bash
xcodebuild -project BangleMe.xcodeproj -scheme BangleMe \
  -destination 'platform=iOS Simulator,name=iPhone 16' test
```

Camera and Vision integration is verified manually on a real device (see Plan 1 Task 12 for the scenario checklist).
